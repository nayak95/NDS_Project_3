Source Code Description:
The source code is under src folder in 'Project 2.zip' and  comprises of 6 .java file:
– 'Runner.java'
This contains the main() function from which the execution starts.
– 'Counter.java'
This is the parent class which implements functions for randomizing elements, randomizing seed array and implementing hash functions.
– 'CountMin.java'
Contains the implementation for Bloom Filter and related functions.
– 'CountSketch.java'
This file contains the implementation for Counting Bloom Filter and related functions.
– 'CountActive.java'
This file contains the implementation for Coded Bloom Filter and related functions.
– 'Flow.java'
This file contains the element class
This contains the Entry class which is used in above files to create Table Entries and Flows.

The folder also contains the 3 output files for each type of Sketch in the following format: output_(type of Sketch). They are as follows:
– 'output_bloom.txt'
– 'output_counting.txt'
– 'output_coded.txt'

These file contains output as per the project specification.
